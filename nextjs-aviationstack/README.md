## Antes de Iniciar
Node version 22.14.0
Next version 15.2.3

## Iniciar

```bash
npm install
npm run dev
# or
yarn dev
```

Abrir [http://localhost:3000](http://localhost:3000) 

## Notas
* El Api de aviationstack no permite usar el search por eso no se incluye
* Se usa un googlemaps embebido para latitud y longitud
* Se usa zustand
* Es responsivo
* Botones con efecto
* Usa axios
